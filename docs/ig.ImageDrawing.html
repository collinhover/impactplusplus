ig.

ImageDrawing

new ImageDrawing()

Image for pixel perfect scaled dynamic drawing.
- this class is intended for use of cached drawings to improve performance, i.e. don't redraw a lot IMPORTANT:do not use this for actual image files, instead use ig.Image.

Author:
  • Collin Hover - collinhover.com

finalize()

Draws the canvas into data at the requested scale ( usually the system scale ) to prepare for final drawing in the game.
- does nothing if system scale is 1 or image drawing is ignoring scale due to ig.ImageDrawing#ignoreScale Tip: because pixel perfect scaling is so performance intensive, try to ignore scale whenever possible.
