ig.

Layer

new Layer()

Layer used to organize entities within game.
- collision checks are handled by layer, i.e. entities cannot collide with entities on another layer.
- pausing game happens on game, layer, and entity levels, and a layer can be set to unpausable. IMPORTANT: for performance, collisions are only checked when something on the layer has changed!
